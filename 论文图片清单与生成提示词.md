## 图3：系统架构图（System Architecture Diagram）

- **插入位置**：第 3 章 → 3.2 系统架构设计（约第 249 \~ 258 行，文字描述"系统架构如下"之后）
- **图片类型**：分层架构图
- **说明**：展示系统的前后端分离六层架构——前端层、API层、服务层、数据访问层、数据库层、AI服务层，以及层间的数据流。

### 生成提示词

> 请生成一张系统分层架构图（System Architecture Diagram），用于一个"基于Spring Boot + Vue的家乡文旅宣传平台"的毕业论文。采用分层架构风格。

层次结构（从上到下）：

- 第一层：前端层（Vue.js + Element UI）—— 用户界面、页面展示、用户交互
- 第二层：API层 / 表现层（Spring Boot Controller）—— RESTful API、请求处理、JWT认证
- 第三层：服务层 / 业务逻辑层（Spring Boot Service）—— 业务逻辑处理、事务管理、AI服务调用
- 第四层：数据访问层（MyBatis Plus Mapper）—— 数据库CRUD操作、分页查询
- 第五层：数据库层（MySQL 8.0）—— 数据持久化存储
- 右侧独立模块：AI服务层（DeepSeek API）—— 智能攻略生成、智能问答

层间关系：

- 前端层 ← HTTP/HTTPS → API层
- API层 ← 方法调用 → 服务层
- 服务层 ← 方法调用 → 数据访问层
- 数据访问层 ← JDBC/SQL → 数据库层
- 服务层 ← HTTP → AI服务层

额外标注关键技术组件：

- Spring Security（安全认证）
- JWT Token（身份令牌）
- Redis（缓存）
- Vue I18n（国际化）

风格：多层架构框图，每层用不同颜色的大矩形表示，层间用箭头连接表示调用方向，整体适合论文排版。
